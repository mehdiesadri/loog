## Quick Start
1. Clone repository
2. Change directory to the cloned repository
3. Create a python virtual environment

    ```python -m venv venv```
4. Activate `venv`

    ```source ./venv/bin/activate```
5. Install Requirements

    ```pipenv install```
6. Run Server

    ```
    cd loog
    python manage.py runserver
    ```
